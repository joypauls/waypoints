# Waypoints [WIP]

A simple, opinonated library for caching temporary results and recording what happened in data science development workflows. Zero dependencies.

---

## Don't tools like MLflow already solve for this?

MLflow is great! So are several others like it out there. They do much more than this tool does. Waypoints is trying to solve a tiny problem to help in the messy experimental phase when projects are iterating fast. While it's safe for production pipelines, that's not what it's built for. You are expected to outgrow it if your work is destined for any kind of serious deployment.
