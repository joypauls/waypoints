# Waypoints

A simple utility for making multi-step Python workflows durable.

📍 No server, no UI, just local files.
📍 Mindless object caching and persistence.
📍 Zero dependenncies.

## Installation

```bash
pip install waypoints
```

```bash
uv add waypoints
```

```bash
poetry add waypoints
```
